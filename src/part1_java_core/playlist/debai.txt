1. Interface Playable
Các phương thức:

void play() – phát nội dung.

void stop() – dừng phát.

int getDuration() – trả về thời lượng (giây).

String getInfo() – trả về thông tin chi tiết.

2. Lớp Media (cha chung)
Thuộc tính:

title (tên)

durationInSeconds (thời lượng)

Getter: getTitle(), getDuration().

3. Lớp Audio (kế thừa Media, implements Playable)
Thuộc tính: artist (ca sĩ/nhóm nhạc).

play(): In ra "Playing audio: '...' by ..."

stop(): In ra "Stopping audio: '...'"

getInfo(): Trả về dạng

Audio[Title: ..., Artist: ..., Duration: ...s]
4. Lớp Video (kế thừa Media, implements Playable)
Thuộc tính: quality (ví dụ: "1080p", "4K").

play(): In ra "Playing video: '...' [quality]".

stop(): In ra "Stopping video: '...'".

getInfo(): Trả về dạng

Video[Title: ..., Quality: ..., Duration: ...s]
5. Lớp Playlist
Bạn cần hoàn thiện các phương thức còn thiếu:

playAll()
In ra thông báo bắt đầu phát playlist.

Dùng vòng lặp duyệt tất cả mục trong playlist.

Với mỗi mục, gọi play().

In ra thông báo kết thúc playlist.

getTotalDuration()
Tính tổng thời lượng tất cả các mục trong playlist.

findLongestItem()
Trả về mục có thời lượng dài nhất.

Nếu playlist rỗng → trả về null.
